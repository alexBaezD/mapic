Mapic , un nuevo concepto en la programación de un PIC

Permite desarrollar de una manera facil, codigo para programar PIC16F887, traduce el codigo a C y utiliza el compilador XC8 de Microchip

Requisitos Para el Correcto Funcionamiento

Tener Instalado Proteus 8,Simulador de Circuitos Tener Instalado XC8 , el compilador de C que proporciona la empresa Microchip
