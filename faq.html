Problemas Comunes y Algunas Soluciones

Para comprender Mapic


Ayuda

¿Por qué no se ejecuta mi programa?

Probablemente no tienes instalado Java
Verificando que java este instalado.
Damos click en el botón de inicio -> Panel de control -> Vista iconos grandes.
Buscamos un logotipo semejante al siguiente.

Damos doble click sobre el icono de java, para verificar la versión de java que tenemos.
Oprimimos el botón de Acerca de...
Y verificamos nuestra versión de java como se muestra en la imagen.

La version de java debe ser mayo o igual a la version 8.

¿Por qué no se compila mi programa?

Falta instalar el compilador XC8
En la instalación no se pudo instalar el compilador XC8. Instale nuevamente el IDE Jasmo Studio y verifique si el error fue corregido.

¿En dónde se guarda mi programa cuando lo termino de compilar?

El programa ya compilado con el código en C está guardado en la dirección que usted eligió al inicio esa dirección es la que el proyecto toma por defecto en dicha dirección es en donde el programa se encuentra.

¿En qué sistemas operativos funciona Mapic?

Esta versión funciona solo en el sistema operativo Windows 7.
