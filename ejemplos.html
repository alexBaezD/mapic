Buenas Prácticas

Para comprender Mapic


Programa Básico
Empezando, a encender un led. Para comprender el uso de Mapic y su Sintaxis.

  • Materiales y Equipo

  • Computadora con Windows 7 o 8
  • Jasmo Studio instalado
  • XC8 Compiler
  • Proteus (ISIS)
  • Protoboard
  • Microcontrolador PIC16F887
  • 8 LED’s
  • 8 Resistencias de 220 ohms
  • 3 Push Button o un dip switch de 4
  • 3 Resistencias de 330 ohms

El siguiente diagrama muestra el circuito básico a utilizar

 

       Fuses

          Setup {
              TA=SALIDA;
              PD=SALIDA;
          }

           Retardo ();

           Ciclo {
               PD0 TOGGLE PD0;
               TimeMS (50000);
           }

           Main {
               TimeMS (1);
           }

void  derizq()
{
	var n:char;
	n=0x80;
	while(1) do
		  PD = n;
			TimeMS(5000);
			Si(n==0x01) Then
				return ;
			No
				n=n<<1;
			Fin
	fin while
}

void izqder()
{
	var n2:char;
	n=0x01;
	while(1) do
			PD = n2;

				return ;
			No
				n=n>>1;
			Fin
	fin while
}

	izqder();
