Buenas Prácticas

Para comprender Mapic


Programa Básico
Empezando, a encender un led. Para comprender el uso de Mapic y su Sintaxis.

  • Materiales y Equipo

  • Computadora con Windows 7 o 8
  • Jasmo Studio instalado
  • XC8 Compiler
  • Proteus (ISIS)
  • Protoboard
  • Microcontrolador PIC16F887
  • 8 LED’s
  • 8 Resistencias de 220 ohms
  • 3 Push Button o un dip switch de 4
  • 3 Resistencias de 330 ohms

El siguiente diagrama muestra el circuito básico a utilizar

 

       Fuses

          Setup {
              TA=SALIDA;
              PD=SALIDA;
          }

           Retardo ();

           Ciclo {
               PD0 TOGGLE PD0;
               TimeMS (50000);
           }

           Main {
               TimeMS (1);
           }

Iniciando

Al crear un proyecto en Jasmo Studio , se crea una carpeta y dentro de ella el archivo, con extensión .mapic

En Jasmo Studio , se presenta el código , a compilar , como se muestra a continuación.

Al Compilar el archivo , en la carpeta del proyecto se crean los archivos correspondientes.

El Resultado Final , seria el siguinte, al traducirlo a lenguaje C

void  derizq()
{
	var n:char;

	fin while

      ciclo{

      		Si(PA == 0b00000010
      		  OR PA == 0b00000100
      		  OR PA == 0b00000110
      		  OR PA == 0b00001000
      		  OR PA == 0b00001010
      		  OR PA == 0b00001100
      		  OR PA == 0b00001110) then

      		No
      			pd0=0;
      		Fin
      	fin while

              caso 0: PD=0b00000000;

              caso 2: PD=0b00000011;
              caso 3: PD=0b00000111;
              caso 4: PD=0b00001111;
              caso 5: PD=0b00011111;
              caso 6: PD=0b00111111;
              caso 7: PD=0b01111111;
              caso 8: PD=0b11111111;
            Fin switch
          fin while
        }

        main{}

La cuenta la mostrara en el puertoD encendiendo los leds que indican la cuenta que lleva, la cuenta deberá tener una duración de 1 seg. Aproximadamente.

          caso 2:
              suma(2);
          caso 3:
              suma(3);
          caso 4:
              suma(4);
          caso 5:
              suma(5);
          caso 6:
              suma(6);
          caso 7:
              suma(7);
          FIN switch
        fin while
      }
